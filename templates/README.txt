试卷 Excel 模板说明

文件名：exam-import-template.xlsx
位置：pc-admin-frontend/templates/

支持列：
1. 题目
2. 选项A
3. 选项B
4. 选项C
5. 选项D
6. 答案
7. 图片地址
8. 排序

填写要求：
1. 必填列：题目、选项A、选项B、答案。
2. 答案只支持 A / B / C / D。
3. 如果答案是 C 或 D，对应选项不能为空。
4. 图片地址可留空；如有图片，请填写可访问的 URL。
5. 排序建议从 1 开始递增；不填时系统也会按行顺序处理。
6. 试卷名称、说明、时长、是否绑定等信息在管理后台页面填写，不在 Excel 中填写。
